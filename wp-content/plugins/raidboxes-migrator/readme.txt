migrate your Site to RAIDBOXES

== Description ==

This tool helps you to migrate your Site to RAIDBOXES. Just insert your RB-Credentials and which BOX we should migrate
your site to, and everything will go automatically :)



== Installation ==

1. Upload `raidboxes-migrator.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. See it under RB-migrator and enter your data, then click start ! :)
